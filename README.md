# Power Zone – Fitness & Wellness Platform 🏋️

Power Zone is a responsive fitness and wellness web platform designed to help users explore workout programs, discover exercise resources, follow nutrition plans, and track their fitness journey.

The project was developed as a web development project using HTML, CSS, and JavaScript, with a focus on responsive design, interactive user interfaces, smooth navigation, and creating a modern fitness-oriented experience.

> 🚧**Project Status: Work in Progress**
>
> The current version represents the implemented stage of the project. Additional workout programs, nutrition plans, user account features, database integration, progress tracking, and personalized features are planned for future development.

---

## Overview

Power Zone aims to provide users with a centralized platform for fitness and wellness resources.

The platform is designed around several main areas:

- Workout and exercise resources
- Training programs
- Nutrition and diet plans
- User profiles and body statistics
- Fitness progress tracking
- Personalized fitness experiences

The long-term goal is to develop Power Zone into a more complete fitness platform where users can choose their goals, select suitable training and nutrition plans, follow their progress, and receive personalized recommendations.

## Current Features

### Home Page

The home page introduces users to the Power Zone platform through a modern fitness-themed interface.

It currently includes:

- A full-screen fitness-themed hero section
- Dark image overlay for a cinematic visual style
- Animated welcome message
- Slideshow-based content sections
- Daily fitness tips
- Latest workout and nutrition programs
- Sports news updates
- Smooth scrolling navigation
- Responsive layout for different screen sizes

The homepage also includes a navigation menu that remains fixed while scrolling through the website.

---

### Animated Slideshow

The homepage includes a slideshow system with animated transitions.

The current sections include:

- Welcome to Power Zone
- Daily Tips
- Latest Programs
- Sport World Updates

The welcome section uses fade-in and slide animations to introduce the platform.

The sports update section is designed to combine a sports-related image with an overlay containing the news content.

The current version uses static sample content as a temporary implementation.

Future versions will include dynamic content that can change based on the latest programs and sports updates.

## Exercises

Each muscle group is presented as an interactive visual card.

The cards include:

- Color transitions

### Planned Exercise Features

Each exercise is planned to include:
